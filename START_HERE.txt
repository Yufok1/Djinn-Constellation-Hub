==================================================

Welcome to the Djinn Constellation Hub!
This package gives you a complete, professional AI constellation system with smart routing, learning, and multi-agent orchestration — all in one easy-to-use bundle.

-------------------------------------
GETTING STARTED (ALL USERS)
-------------------------------------

1. **Unzip this package** to a folder on your computer.

2. **Install Ollama** (required for running models)
   - Download and install from https://ollama.ai
   - Make sure `ollama` is in your PATH (run `ollama --version` to check)

3. **Run the One-Click Installer**
   - **Windows:** Double-click `install_djinn_federation.bat`
   - **Linux/Mac:**
     ```bash
     chmod +x install_djinn_federation.sh
     ./install_djinn_federation.sh
     ```
   - This will:
     - Pull all required base models
     - Build all custom Djinn models (Council, IDHHC, Companion, Federation)
     - Show next steps for launching the system

4. **Launch the Constellation Hub**
   - **For the complete experience:**
     - **Windows:** Double-click `launch_constellation_complete.bat`
     - This launches the full Constellation Hub with intelligent routing
   - **For individual model testing:**
     ```bash
     ollama run Yufok1/djinn-federation:constellation
     ollama run Yufok1/djinn-federation:council
     ollama run Yufok1/djinn-federation:idhhc
     ollama run Yufok1/djinn-federation:companion
     ```

5. **Test the System** (Optional)
   - Run `python test_constellation_fix.py` to verify everything works
   - This tests encoding compatibility and model availability
