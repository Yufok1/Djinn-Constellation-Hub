-------------------------------------
SYSTEM REQUIREMENTS
-------------------------------------
- **RAM**: 8GB+ minimum (32GB recommended for all models)
- **Storage**: 50GB+ for all federation models
- **Ollama**: Latest version installed (https://ollama.ai/download)
- **Python**: 3.8+ for Constellation Hub system
- **OS**: Windows, Linux, or macOS

-------------------------------------
QUICK INSTALLATION (ALL USERS)
-------------------------------------

1. **Install Ollama** (required for running models)
   - Download and install from https://ollama.ai
   - Make sure `ollama` is in your PATH (run `ollama --version` to check)

2. **Clone the Repository**
   ```bash
   git clone https://github.com/Yufok1/Djinn-Constellation-Hub.git
   cd Djinn-Constellation-Hub
   ```

3. **Run the One-Click Installer**
   - **Windows:** Double-click `install_djinn_federation.bat`
   - **Linux/Mac:**
     ```bash
     chmod +x install_djinn_federation.sh
     ./install_djinn_federation.sh
     ```
   - This will install all four federation models (total ~40GB)

4. **Test the Installation** (Optional but Recommended)
   ```bash
   python test_constellation_fix.py
   ```
